# streamdeck-setup

Personal Stream Deck+ setup.

## Projects

- **[logitech-battery-tracker](logitech-battery-tracker/)** — Stream Deck plugin that shows a
  Logitech wireless device's battery on a key: level fill (smooth/tiers), optional percentage,
  and a charging tint on the outline. Reads battery over HID++ (event-driven), with a Logitech
  G HUB fallback. See its [README](logitech-battery-tracker/README.md).
